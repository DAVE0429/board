## 프로젝트 소개
SpringBoot와 JPA를 이용하여 게시판 만들기 프로젝트입니다.
현재로선 프론트엔드가 있다는 가정하에 백엔드로 api을 작성하였습니다.
추가로 프론트엔드 작업도 해보려고 합니다.
